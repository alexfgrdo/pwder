# Pwder

> Simple password generator developed with Python.

## Getting Started

A step by step series of examples that tell you how to get this script running.

### Prerequisites

- [Python](https://python.org)

### Installing

`git clone https://github.com/alexfgrdo/pwder.git`

### Running

`python pwder.py`
